the proliferation of digital images in unsecured network channels, images are vulnerable to diverse attacks. Cryptographic encryption techniques with hybrid approaches must be incorporated to protect the digital images. The proposed methodology involves hybrid approach that uses  Hexa-Affine Key Deterministic Modular Permutation (HAKDMP) along with Quantum inspired Basis Matching Key Extraction (BMKE) techniques incorporated with pixel transformation. This technique adds an additional security layer to the images and protects from diverse quantum attacks and brute-force attacks. This framework is evaluated with security measures and experimental results show that it provides higher level of security and resilience with attacks.
